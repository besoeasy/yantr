Frequently asked questions

How is this different from UmbrelOS?

UmbrelOS is a complete operating system that replaces your current OS. Yantra is an application that runs on top of your existing OS (Ubuntu, Debian, macOS, etc.) using Docker. This means you can keep using your computer as normal while running Yantra in the background.

What hardware do I need?

Any machine that can run Docker can run Yantra. This includes old laptops, Raspberry Pis, mini PCs, massive servers, or even your MacBook. Requirements depend on the apps you want to run (e.g., AI agents need more RAM).

Is it really open source?

Yes, 100%. You can inspect every line of code on GitHub. We do not track you, and your data stays on your machine.

Can I expose apps to the internet?

By default, Yantra is designed for local network access. However, since it's just Docker, you can easily set up Cloudflare Tunnels (via the Cloudflared app in our store) or a reverse proxy like Nginx Proxy Manager to safely expose services.
